Dataset :- 
We use the dataset 'Spotify Charts' from Kaggle, with a file size of 3.48 GB.
It has 26,173,514 observations and 9 columns, including title, rank, date, artist, url, region, chart, trend, and stream. This dataset contains the top 200 streamed tracks on Spotify every day from Jan 1 2017 to Dec 31 2021
